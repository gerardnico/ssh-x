# SSH-X  - A collection of SSH utilities


## About

`SSH-X` is a collection of utilities around `OpenSSH` 

## List

* [ssx](docs/bin/ssx) - A ssh wrapper to manage common ssh operations without knowing all whistle.
  * [ssh-key](docs/bin/ssh-key.md) - Manage your keys
  * [ssh-agent-init](docs/bin/ssh-agent-init.md) - Start SSH agent if not started and load private keys
* [ssh-aut-proxy-command](docs/bin/ssh-auth-proxy-command.md) - An utility to load automatically your private keys from your public key as identity
* [ssh-askpass-prompt](docs/bin/ssh-askpass-prompt.md) - A `ASKPASS` implementation to ask a secret
