# SSH-X  - A collection of SSH utilities


## About

`SSH-X` is a command line utility called `ssx` that hosts a collection of utilities around `OpenSSH`

## List

* [ssx](docs/bin/ssx) - the main cli entry
* [ssh-aut-proxy-command](docs/bin/ssh-auth-proxy-command.md) - Proxy command to load private keys from public key
* [ssh-askpass-prompt](docs/bin/ssh-askpass-prompt.md) - A `ASKPASS` implementation to ask a secret
