% ssh-x-key-passphrase(1) Version Latest | Manage your key passphrase


# NAME

`ssh-x-key-passphrase` - A command to set or delete OpenSSH Private key passphrase


## SYNOPSIS

```bash
sshx key passphrase command file
```

where `command` can be:

* `delete`      - Delete the passphrase
* `set`         - Set a passphrase
