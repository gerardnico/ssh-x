% ssh-x(1) Version Latest | SSH Utilities and extensions


# NAME

`ssh-x` - A cli of SSH Utilities and eXtensions


## SYNOPSIS

```bash
ssh-x command options
```

where `command` can be:

* `key`   - Key operation
* `info`  - List the state of SSH Key and Agent


## SSX Alias

`SSH-X` is also known as `ssx`. You can create an alias in your `.bashrc`

Example:
```bash
alias ssx="ssh-x"
```
